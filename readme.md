### Lamp docker-compose 

- Based on official images
- Debian jessie
- php 5.6.29
- mariadb 10.1


## How to use

```bash
git clone https://github.com/dannielarriola/docker-compose-lamp.git
cd docker-compose-lamp/
docker-compose up -d
```
